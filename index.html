HTML 5

Es la versión más reciente y un conjunto de tecnologías web que incluye HTML, CSS y JavaScript, utilizada para estructurar y presentar contenido web.

Estándar de lenguaje que permite aplicar estilos y dar una presentación visual a documentos web escritos en HTML, manteniendo el contenido separado del diseño.

Lenguaje de scripting versátil que permite crear contenido dinámico y funcionalidades interactivas en páginas web y aplicaciones. Se ejecuta tanto en el navegador (lado del cliente).
